# WebSocket + TLS (HTTP/1.1)

# Requirements
## Server
- xray-core  stable version (v25.8.x or more)
- optional but recommended, a panel for xray like 3x-ui and remnawave
- nginx latest version (1.2x)
  - ~~please note, apt package manager ships v1.24.x as latest version, make sure to change the repo to nginx`s one~~ we dont need http2, not relevant
- a domain which you own
  - it points to your vps ip adress
- valid ssl certificates
## Client 
- a client
  - can be based on sing box, mihomo
## Miscellaneous 
- a working brain
# Why use this setup?
- REALLY stable
- good compatibility
  - most clients support unlike XHTTP 
- stealthy
- decent performance
